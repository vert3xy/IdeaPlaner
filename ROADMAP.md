# IdeaPlaner Roadmap

Этот документ содержит план развития продукта, идеи по улучшению и технический бэклог. План разбит на логические модули для удобства ведения мини-релизов.

## 🚦 Статус проекта: [v1.0-MVP Running]
- ✅ Базовая авторизация (JWT)
- ✅ Нормализованная БД (Категории, Атрибуты, Идеи)
- ✅ Динамическая отрисовка фильтров и форм на основе метаданных
- ✅ Базовый мобильный интерфейс

---

## План ближайших релизов

### v1.1: Визуал и UX
- ✅ **Master-Detail Layout:** Переход от модальных окон к двухпанельному интерфейсу (список слева, детали справа).
- [ ] **Compact List View:** Режим отображения "компактная строка" для быстрой прокрутки большого количества идей.
- [ ] **Frontend Routing:** Поддержка History API — возможность делиться ссылкой на конкретную карточку.
- [ ] **Skeleton Loaders:** Красивая индикация загрузки данных вместо резкого появления карточек.
- [ ] **Dark Mode:** Ночная тема для комфортного планирования в вечернее время.

### v1.2: Кибербезопасность
- [ ] **Audit Logger:** Реализация таблицы аудита в БД. Фиксация действий пользователей (кто, когда и какой объект изменил).
- [ ] **RBAC (Role-Based Access Control):** Роли `Owner` и `Partner`. Разграничение прав на удаление и редактирование глобальных категорий.
- ✅ **Input Sanitization:** Защита от XSS-атак при вводе данных в JSON-атрибуты.
- [ ] **Rate Limiting:** Ограничение количества запросов к API для защиты от брутфорса и перегрузки.
- [ ] **Nginx + SSL:** Настройка зашифрованного соединения (HTTPS) и скрытие приложения за реверс-прокси.

### v1.2.1: Визуал и UX 2.0
- [ ] **Skeleton Loaders:** Плавная индикация загрузки контента.
- [ ] **Custom Toasts:** Замена системных alert() на стильные уведомления.
- [ ] **Mobile Navigation:** Нижняя панель управления для работы "одной рукой".

### v1.2.2: Рефакторинг и Архитектура
- [ ] **Alembic Integration:** Профессиональное управление миграциями БД.
- [ ] **Service Layer:** Вынос бизнес-логики из эндпоинтов для упрощения тестирования.
- [ ] **JS Modularization:** Разбиение фронтенд-логики на независимые модули.
- [ ] **Unit Testing:** Покрытие тестами (Pytest) критических узлов (Auth, CRUD).

### v1.3: Панель управления
- [ ] **Dashboard Page:** Визуализация статистики (всего идей, воронка статусов, активность по категориям) с использованием графиков (Chart.js).
- [ ] **Users Management (RBAC):** Страница управления доступом. Создание пользователей, назначение ролей (Admin, Editor, Viewer), управление сессиями.
- [ ] **Audit Logs Page:** Просмотр истории действий в системе. Фильтрация логов по пользователю, дате и типу события (кибербез-мониторинг).
- [ ] **Categories Editor:** Интерфейс для создания и редактирования категорий прямо из приложения (название, иконка, цвет).
- [ ] **Attributes Manager:** Страница управления атрибутами и их связями с категориями. Позволяет настраивать, какие поля должны быть у "Кино", а какие у "Путешествий".

### v1.4: Функциональность
- [ ] **The Matcher:** Алгоритм "Совпадений". Возможность лайкать идеи; при мэтче обоих партнеров идея переходит в статус "Запланировано".
- [ ] **Randomizer (Dice):** Кнопка случайного выбора идеи из текущего отфильтрованного списка.
- [ ] **Attachments:** Поддержка хранения ссылок на внешние

## v 1.5: Автоматизация и Данные
- [ ] **Smart Link Parser:** Автозаполнение карточек при вставке ссылок (Кинопоиск, Steam, Youtube).
- [ ] **Geo-Suggester:** Интеграция с картами для быстрого выбора адресов и построения маршрутов.
- [ ] **Telegram Sync:** Бот для быстрого добавления идей в базу через мессенджер.

### v 1.6: Эмоции и Воспоминания
- [ ] **Impressions System:** Поле для отзывов и 5-звездочный рейтинг для выполненных идей.
- [ ] **Photo Gallery:** Хранение фотографий, привязанных к реализованным планам.
- [ ] **Anniversary Recap:** Автоматическая подборка "Что мы делали в этот день год назад".

## Идеи для развития (Growth)
- **Marketplace категорий:** Возможность делиться наборами атрибутов (например, "Идеальный набор для любителей настолок").
- **Gamification:** Система ачивок за выполненные планы ("Киноманы 80 уровня", "Путешественники года").
- **Public API:** Для интеграции со сторонними сервисами календарей (Google Calendar, Apple Calendar).
